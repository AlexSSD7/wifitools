# Installation
Requirements:
* Kali Linux
* Python 3

Open your terminal and type:
<pre>git clone https://github.com/AlexSSD7/wifitools.git
cd wifitools
python3 setup.py
</pre>
After that, you can open wifitools by typing
<pre>wifitools</pre>
