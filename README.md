# Tool to perform different Wi-Fi pranks
Uses Python 3 and can deauth and spawn a lot of fake APs.

# Installation
Requirements:
* Kali Linux
* Python 3

<pre>git clone https://github.com/AlexSSD7/wifitools.git
cd wifitools
python3 setup.py
</pre>
After that, you can open wifitools by typing
<pre>wifitools</pre>
